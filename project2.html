Project 2

This project was created to house all of my other projects. I made it to learn how to connect my URL to Github. I used HTML, CSS, JavaScript and Bulma to create it. The software I used were: Git, Github, Github Pages, Atom, Command Prompt, FireFox, Godaddy. The easy part was using bulma as I already know how to use it. It was hard adding content to the project pages. If I were to do it again I would add more features and content. I learnt how to upload code to a Github repository and connect a GoDaddy URL to a Github Page site.
